# 积分与微分的本质

	作为上大学接触的新的计算，积分与微分相比于四则运算要更难以习惯。
	
	高数老师说：“很多初学者会犯的错误都源于对概念的忽视。其实这和你们接受的初等数学教育有关。”
	
	“你们习惯了四则运算，那些运算法则很自然地被你们运用，你们无需考虑基本法则，只需要闷头算就好了。于是你们以同样方式对待高等数学，很多忽略运算法则的操作就出现了。”
	
所以我觉得停下来捋一捋积分和微分，对我来说还是有点重要的。

## 1.不定积分

若函数 $F(x)$的导数是 $f(x)$（即 $F′(x)=f(x)$），则称$F(x)$是 $f(x)$的一个原函数。

而不定积分定义为“全体原函数的集合”：

$\int f(x)dx = F(x) + C \qquad(C为积分常数)$

不定积分是微分的“逆过程”

——微分是“求变化率（导数）”，不定积分是“由变化率还原函数”。

但这个定义高度抽象：

- 只说“全体原函数”，却未解释“原函数从何而来？”
- “为什么原函数之间差一个常数？” 
- “原函数一定存在吗？” 

为了解决以上，我们来唠唠积分上限函数——

## 2.积分上限函数

积分上限函数，它使得定积分的计算变得简洁；它是全体原函数的一个实例，是不定积分的具象化。
{ #f61826}


设$f(t)$在区间 $[a,b]$ 上连续，取固定起点 $a$，让上限 $x$ 在 $[a,b]$ 内变动，则定义函数：

$\Phi(x) = \int_{a}^{x} f(t) dt \quad(a\leq x \leq b)$

这就是积分上限函数。

### 积分上限函数的分析性质 

#### 连续性

无论 $f(t)$ 是“连续”还是“可积”，$Φ(x)$ **始终连续**（**这是定积分“区间可加性”的保证**）

#### 可导性

若 $f(x)$ 连续，则 $Φ(x)$ 可导，且导数恰好是$f(x)$ 本身：

$\Phi'(x) = \frac{d}{dx} \int_a^x f(t)dt = f(x)$

严格证明
若 f(t)在 $[a,b]$ 上可积，则 $Φ(x)=\int_a^xf(t)dt$在 $[a,b]$ 上连续

分割区间：任取 $x∈[a,b]$和增量 $Δx$，使得 $x+Δx∈[a,b]$

积分差值：$ΔΦ=Φ(x+Δx)−Φ(x)=∫_{x}^{x+Δx}f(t)dt$

利用有界性：设 $∣f(t)∣≤M$，则：

$∣ΔΦ∣≤∫_x^{x+Δx}f(t)dt ≤ ∫_x^{x+Δx}∣f(t)∣dt≤M∣Δx∣$

极限分析：$当 Δx→0时，M∣Δx∣→0$，故：

$\lim_{\Delta x \rightarrow 0}ΔΦ=0$

### 积分上限函数是不定积分（原函数）的具体化

不定积分函数可直接由积分上限函数表示为

$\int f(x)dx = \Phi(x) + C = \int_a^x f(t)dt + C$

只要 $f(x)$连续，积分上限函数天然就是它的一个原函数

## 3.定积分

定积分的定义基于黎曼和的极限：

$\int_a^b f(t)dt = \lim_{\lambda\rightarrow 0} \sum_{i=1}^n f(\xi_i) \Delta t_i$

*其中 λ是区间分割的最大长度，ξ​是子区间的任意点，$\Delta t_i$是子区间长度*

通过积分上限函数，定积分的计算变得简洁——牛顿-莱布尼茨公式揭示了定积分与不定积分（原函数）的联系：

若 F(x)是 f(x)的一个原函数，则：

$\int_a^b f(x)dx = F(b) - F(a)$

因此，定积分的计算就变成了**原函数的差值**。
