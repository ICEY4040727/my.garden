$\displaystyle = \int \frac {1}{t^2 + \frac{1}{t^2} - 1}\ d\ \ (t-\frac{1}{t})$

$\displaystyle= \int \frac{1}{(t-\frac{1}{t})^2 + 1}\ d\ \ (t-\frac{1}{t})$
