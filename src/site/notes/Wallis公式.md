# 内容

## **当$m$为正偶数，设 $m = 2n (n∈ Z_+)$

### $\displaystyle\int_0^{\frac{\pi}{2}} \sin^{2n} x dx = \frac{(2n - 1)!!}{(2n)!!} \cdot \frac{\pi}{2}$

## 当m为正奇数，设$m=2n+1(n∈ N)$

### $\displaystyle\int_0^{\frac{\pi}{2}} \sin^{2n+1} x dx = \frac{(2n)!!}{(2n + 1)!!}$
