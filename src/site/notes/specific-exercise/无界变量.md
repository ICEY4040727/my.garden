# $n * sin(n)$
{ #da0ce0}



对于  $n * sin(n)$：

 $n$  是无穷大。

 $sin(n)$  是一个有界变量（值在  [-1, 1]  之间），但它不会趋于0。

所以  $n * sin(n)$  的绝对值会随着  n  的增大而无限振荡

其结果是无界变量（更具体地说，是振荡无界），而不是无穷大或有界
