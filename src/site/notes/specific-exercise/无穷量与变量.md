#### 1. 当$n\rightarrow \infty$ 时，$n \sin{\frac{2}{n}}$ 是一个（ ）
- 无穷小
- 无穷大
- 无界变量
- 有界变量
 $$当n \to \infty 时，\frac{2}{n} \to 0$$$$利用等价无穷小代换 \sin x \sim x（当 x \to 0 时$$得
$$n \sin \frac{2}{n} \sim n \cdot \frac{2}{n} = 2.$$


#### ==为什么不可以认为这是无穷大乘有界函数结果是无穷大？==

1. 我们通常说“无穷大乘以有界函数”时，

   这个“**有界函数**”指的是一个**固定**的函数，比如  *sin(n)*  或  *cos(n)* 。

   这些函数的值始终在  [-1, 1]  之间震荡，不会随着  n  趋于无穷而趋于0。


2. 本题中的  sin(2/n)  为什么不是“固定的有界函数”？

   在  n * sin(2/n)  中， sin(2/n)  这个因子是依赖于n的，并且当  n → ∞  时， 2/n → 0 。

   因此， sin(2/n)  不是一个固定的有界函数，而是一个无穷小量（因为它趋于0）。

#### 无穷与无界变量的区别--变化趋势的单调与否

无穷大一定是无界变量：
无穷大要求“绝对值无限增大”，自然满足“无界”（没有固定的界）。

无界变量不一定是无穷大：
无界变量只需“绝对值可超过任意正数”，但不需要“单调无限增大”，甚至可以震荡。
