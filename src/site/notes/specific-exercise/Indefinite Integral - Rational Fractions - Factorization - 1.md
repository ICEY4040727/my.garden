# $\frac{A}{(x-a)^k}$ 型 


>[!info] 添项减项化归

$\displaystyle I = \int \frac{1}{(x^2 + a^2)^2}dx$

$\displaystyle = \frac{1}{a^2} \int \frac{a^2 + x^2 -x^2} {(a^2 + x^2)^2}dx$

$\displaystyle = \frac{1}{a^2} \int \frac{1} {(a^2 + x^2)}dx + \int \frac{-x^2} {(a^2 + x^2)^2}dx$

# $\frac{x^2}{(x-a)^k}$ 型

>[!info] 分部积分以给分母降次
{ #36ff0c}


$\displaystyle = \int \frac{x^2} {(a^2 + x^2)^2}\ \ dx$
{ #92d7e4}


$\displaystyle =\frac{1}{2}\int {x} \ \ \ d \frac{1} {(a^2 + x^2)}$

$\displaystyle =\frac{1}{2}x \cdot \frac{1} {(a^2 + x^2)} - \frac{1}{2}\int \frac{1} {(a^2 + x^2)} \ \ \ d {x}$

$\displaystyle =\frac{1}{2}\ [\frac{x} {(a^2 + x^2)} - \frac{1}{a}{\arctan\frac{x}{a}}]$
