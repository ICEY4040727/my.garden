# 共轭裂项
{ #2daebd}


$\displaystyle I = \int \frac{1}{1-x^4} \  dx$

$\displaystyle = \int \frac{1}{1+x^2} \cdot \frac{1}{1-x^2}\ dx$

>[!info] $\int \frac{1}{x^2 - a^2 } \ dx = \frac{1}{2a}\ \ln |\frac{x-a}{x+a}| + C$

$\displaystyle = \frac{1}{2} \int \frac{1}{1+x^2} \ +\  \frac{1}{1-x^2}\ dx$

# 三角函数同乘共轭，减分母项数
{ #8a000f}


$\displaystyle\int\frac{1}{1+\cos x}\ dx$

$\displaystyle =\ \int \frac{1-\cos x}{(\sin x)^2}\ dx$

$\displaystyle=\int(\csc x)^2 dx-\int\frac{\cos x}{(\sin x)^2}\ dx$

$\displaystyle =-\cot x+\frac{1}{\sin x}+C$
